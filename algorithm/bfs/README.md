# BFS (Breadth-First Search, 너비 우선 탐색)

시작 노드로부터 인접한 모든 노드를 먼저 탐색한 후, 다음 레벨의 노드들을 차례대로 탐색하는 방식입니다.

## 핵심 원리

1. 시작 노드를 **Queue**에 넣고 **방문 처리**합니다.

2. 큐에서 노드를 꺼내어 해당 노드의 인접 노드 중 **방문하지 않은** 노드를 모두 큐에 넣고 방문 처리합니다.

3. 큐가 빌 때까지 위 과정을 반복합니다.

### 특징

+ **자료구조:** `Queue` (FIFO)를 사용합니다.

+ **최단 거리 보장:** 가중치가 없는 그래프에서 시작점에서 특정 노드까지의 **최단 경로**를 찾는 데 최적화되어 있습니다.

+ **방문 체크:** 큐에 넣을 때 반드시 방문 처리를 하여 중복 탐색을 막아야 합니다.

## 핵심 로직 (Queue BFS)

BFS는 큐(Queue)를 이용해 현재 노드와 가까운 노드부터 순차적으로 탐색합니다.

+ **구현 사례:** [BFS.java](./BFS.java)

```java
/**
 * V : 시작 노드
 * N : 전체 정점의 개수
 * visited : 방문 여부 확인 배열
 */
public static void BFS(int V, int N) {
    Queue<Integer> que = new LinkedList<>();
    
    que.offer(V); // 시작 노드 삽입
    visited[V] = true; // 방문 처리
    System.out.print(V + " ");

    while (!que.isEmpty()) {
        int current = que.poll(); // 큐에서 하나를 꺼냄

        for (int i = 1; i <= N; i++) {
            // 방문하지 않았고 연결된 노드라면
            if (!visited[i] && arr[current][i] == 1) {
                que.offer(i); // 큐에 추가
                visited[i] = true; // 추가함과 동시에 방문 처리
                System.out.print(i + " ");
            }
        }
    }
}
```

## 시간 복잡도

+ **인접 리스트:** $O(V + E)$

+ **인접 행렬:** $O(V^2)$

+ **공간 복잡도:** $O(V)$ (큐에 담기는 노드 수만큼 차지)

## 실전 활용

+ **최단 경로 탐색:** 두 노드 사이의 최단 거리나 최소 이동 횟수를 구할 때 (예: 미로 최단 탈출)

+ **네트워크 탐색:** 특정 노드와 인접한 이웃들을 먼저 찾아야 할 때

+ **가중치가 동일한 그래프:** 최단 거리를 $O(V+E)$에 구할 수 있음
