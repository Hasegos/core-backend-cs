# DFS (Depth-First Search, 깊이 우선 탐색)

그래프의 시작 노드에서 한 경로를 따라 최대한 깊게 탐색한 후, 더 이상 갈 곳이 없으면 뒤로 돌아와 다른 경로를 탐색하는 알고리즘입니다.

## 핵심 원리

1. 탐색 시작 노드를 **방문 처리**합니다.

2. 현재 노드와 연결된 인접 노드 중 **방문하지 않은** 노드가 있다면 그 노드로 이동하여 다시 탐색을 시작합니다. (재귀 호출)

3. 방문하지 않은 인접 노드가 없다면 이전 단계로 돌아갑니다.

### 특징

+ **자료구조:** `Stack` 또는 **재귀 함수**를 사용합니다.

+ **방문 체크:** 무한 루프를 방지하기 위해 `visited[]` 배열이 반드시 필요합니다.

+ **장점:** 경로의 특징을 저장해야 하는 문제(예: 특정 조건을 만족하는 경로 찾기)에 유리합니다.

## 핵심 로직 (Recursive DFS)

DFS는 재귀 호출을 이용해 방문하지 않은 인접 노드를 끝까지 파고듭니다.

+ **구현 사례:** [DFS.java](./DFS.java)

```java
/**
 * V : 현재 방문 중인 노드
 * N : 전체 정점의 개수
 * visited : 방문 여부 확인 배열
 * arr[][] : 인접 행렬 (연결 상태 저장)
 */
public static void DFS(int V, int N) {
    // 1. 현재 노드 방문 처리
    visited[V] = true;
    System.out.print(V + " ");

    // 2. 인접한 노드 확인
    for (int i = 1; i <= N; i++) {
        // 방문하지 않았고, 연결되어 있다면(arr[V][i] == 1)
        if (!visited[i] && arr[V][i] == 1) {
            DFS(i, N); // 더 깊게 탐색 (재귀)
        }
    }
}
```

## 시간 복잡도

+ **인접 리스트:** $O(V + E)$

+ **인접 행렬:** $O(V^2)$

+ **공간 복잡도:** $O(V)$ (최악의 경우 스택 깊이만큼 차지)

##  실전 활용

+ **미로 찾기:** 모든 경로를 탐색하며 목표지점까지의 경로를 찾을 때

+ **사이클 판별:** 그래프 내에 순환 구조가 있는지 확인할 때

+ **백트래킹:** 가능한 모든 경우의 수를 탐색해야 할 때
