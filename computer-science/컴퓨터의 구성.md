## 하드웨어의 4대 핵심 부품

컴퓨터 하드웨어는 시스템 버스를 통해 연결되어 데이터를 주고받습니다.

### ① 중앙처리장치 (CPU)

+ **역할:** 컴퓨터의 두뇌로서 프로그램의 명령어를 해석하고 실행합니다.

+ **구성 요소:**

    + **ALU (산술논리연산장치):** 비교와 연산을 담당하는 계산기 역할을 합니다.

    + **제어장치:** 제어 신호를 보내 명령어를 해석하고 부품들을 관리합니다.

    + **레지스터:** CPU 내부의 작고 빠른 임시 저장 장치입니다.

### ② 주기억장치 (Memory - RAM, ROM)

+ **특징:** 실행 중인 프로그램(프로세스)의 명령어와 데이터를 일시적으로 저장합니다.

+ **주소(Address):** 저장된 값에 효율적으로 접근하기 위해 위치 정보를 사용합니다.

### ③ 보조기억장치 (SSD, HDD)

+ **특징:** 주기억장치보다 속도는 느리지만, 데이터를 영구적으로 보관할 수 있습니다.

### ④ 입출력장치

+ **입력 장치:** 키보드, 마우스 등 외부 자료를 컴퓨터 내부로 입력합니다.

+ **출력 장치:** 모니터, 프린터 등 처리된 결과를 외부로 표현합니다.

## 시스템 버스 (System Bus)

하드웨어 구성 요소를 물리적으로 연결하여 데이터와 신호를 전달하는 통로입니다.

+ **데이터 버스:** 명령어와 데이터를 전달하는 양방향 통로입니다.

+ **주소 버스:** CPU가 주소 정보를 전달하는 단방향 통로입니다.

+ **제어 버스:** 제어 신호(읽기/쓰기 등)를 전달하는 양방향 통로입니다.

## 데이터의 단위 및 처리

컴퓨터가 이해하는 가장 작은 단위는 <strong>비트(bit)</strong>이며, 0과 1로 정보를 표현합니다.

### 데이터 정보 단위 표

| 단위 | 용량 | 비고 |
| :--- | :--- | :--- |
| **1 Byte** | $8$ bit | 영문자 1글자를 저장하는 기본 단위 |
| **1 kB** | $10^3$ Byte | $1,000$ Byte (전통적인 SI 단위) |
| **1 MB** | $10^6$ Byte | $1,000,000$ Byte |
| **1 GB** | $10^9$ Byte | $1,000,000,000$ Byte |
| **1 TB** | $10^{12}$ Byte | $1,000,000,000,000$ Byte |

### 처리 단위: 워드(Word)

+ **정의:** CPU가 한 번에 처리(연산/이동)할 수 있는 데이터의 크기입니다.

+ **분류:** 하프 워드($1/2$), 풀 워드($1$배), 더블 워드($2$배)

+ **예시:** **64비트 운영체제**는 한 번에 64비트 데이터를 처리하고 가져올 수 있음을 의미합니다.
