# 📂 Data Structure (자료구조)

컴퓨터 프로그래밍의 기초가 되는 주요 자료구조들의 원리를 이해하고, Java를 통해 직접 구현한 프로젝트입니다.<br>
각 항목을 클릭하면 해당 자료구조의 상세 README와 구현 코드로 이동합니다.

[개념 모음으로 돌아가기](../README.md)

## 📑 목차 (Contents)

### 1. 선형 자료구조 (Linear Data Structures)

데이터가 일렬로 나열된 형태의 자료구조입니다.

+ [**Stack (스택)**](./stack/README.md) - LIFO(후입선출) 원리 및 함수 콜 스택 이해

+ [**Queue (큐)**](./queue/README.md) - FIFO(선입선출) 원리 및 프로세스 대기열 이해

### 2. 비선형 자료구조 (Non-Linear Data Structures)

계층적 또는 망형 구조를 가지는 자료구조입니다.

+ [**Tree (트리)**](./tree/README.md) - 노드 기반의 계층적 구조 및 이진 트리 순회(Traversal) 학습

+ [**Trie (트라이)**](./trie/README.md) - 문자열 검색 최적화를 위한 접두사(Prefix) 공유 구조 학습

### 3. 특수 자료구조 (Specialized Data Structures)

효율적인 탐색과 저장을 위한 구조입니다.

+ [**Hash Table (해시 테이블)**](./hashTable/README.md) - Key-Value 쌍의 저장 및 Separate Chaining 충돌 해결 전략 구현

## 🛠 기술 스택 (Tech Stack)

* **Language:** Java

* **IDE:** VS Code / IntelliJ

* **Focus:** 자료구조의 직접 구현(Implementation)을 통한 동작 원리 심화 이해

## 🔍 학습 포인트 (Quick Summary)

+ **직접 구현:** 라이브러리 사용에 그치지 않고 노드(Node) 구조부터 직접 설계

+ **성능 분석:** 각 자료구조별 삽입/삭제/탐색의 시간 복잡도($O$) 이해
